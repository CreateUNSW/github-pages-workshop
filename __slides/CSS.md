## CSS controls how your site looks

* font size
* font colours
* text-align
* links and hover styles
* margins and paddings
* height, width, sizes
* background colours
* shadow
* and more...

Full documentation [here](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference)

In the form:

``` CSS
selector {
    property: value;
}

selector other {
    other-property: other-value;
}
```
