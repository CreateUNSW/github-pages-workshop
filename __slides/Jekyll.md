## Preparations

You can only run jekyll after installing Ruby.

## Front Matter

```
---
property: value
---
```

Can leave it blank, but needed for Jekyll to pick up the file.

## How Jekyll applies to html and markdown files

Jekyll will convert and build each html and markdown files into generated stand-alone pages.

## Commands

`bundle exec jekyll new`

`bundle exec jekyll serve`

`bundle exec jekyll build`
